### Color Palette
| Element | Color | Usage |
|---------|-------|-------|
| **Primary Dark** | `#0a1628` | Background base |
| **Deep Ocean** | `#1a2942` | Cards, sections |
| **Midnight Blue** | `#0f2439` | Hover states |
| **Electric Cyan** | `#00d9ff` | Primary accent |
| **Neon Blue** | `#00a8cc` | Secondary accent |
| **Premium Gold** | `#ffd700` | Highlights, badges |

### Shadow System
| Type | Specification | Usage |
|------|---------------|-------|
| **Card Shadow** | `0 10px 40px rgba(0,0,0,0.3)` | Cards, containers |
| **Premium Shadow** | `0 20px 60px rgba(0,0,0,0.5)` | Featured elements |
| **Glow Shadow** | `0 0 40px rgba(0,217,255,0.3)` | Hover effects |
| **Glass Shadow** | `0 8px 32px rgba(0,0,0,0.37)` | Glass elements |

### Gradient Library
```css
/* Primary Background */
linear-gradient(135deg, #0a1628 0%, #1a2942 50%, #0f2439 100%)

/* Accent Buttons/Elements */
linear-gradient(135deg, #00d9ff 0%, #00a8cc 100%)

/* Premium Cards */
linear-gradient(135deg, #141e30 0%, #243b55 100%)

/* Glow Overlay */
linear-gradient(135deg, rgba(0,217,255,0.1) 0%, rgba(0,168,204,0.1) 100%)

/* Radial Glow (Hero) */
radial-gradient(circle at 20% 50%, rgba(0,217,255,0.1) 0%, transparent 50%)
```
